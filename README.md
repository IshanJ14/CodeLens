# CodeLens

CodeLens is a full-stack competitive programming analytics platform built with FastAPI and React.

## Features

- Codeforces profile analytics
- Rating history visualization
- Contest performance tracking
- Problem tag analysis
- Submission activity heatmap

## Tech Stack

### Backend
- FastAPI
- Python
- HTTPX

### Frontend
- React
- Vite
- Axios
